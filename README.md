# 👥 KeenKeeper — Keep Your Friendships Alive

Your personal shelf of meaningful connections. Browse, tend, and nurture the relationships that matter most.

---

## 🛠️ Technologies Used

- **Next.js** — Framework and App Router
- **React** — UI components and hooks
- **Tailwind CSS** — Styling and responsive design
- **DaisyUI** — Component library
- **Recharts** — Pie chart for analytics


---

## ✨ Key Features

1. **Friendship Tracking** — Every friend has a color-coded status (On Track, Almost Due, Overdue) so you always know who needs attention

2. **Quick Check-In** — Log a Call, Text, or Video interaction from any friend's detail page in one click — saved to your timeline instantly with a toast notification

3. **Timeline with Analytics** — Full interaction history with filter by type (Call / Text / Video) and a visual pie chart on the Stats page showing your interaction breakdown
